# Lending Portfolio KPI Dashboard

An interactive credit portfolio performance dashboard built with 
Streamlit and Plotly — simulating the kind of management reporting 
tool used by credit analytics teams in fintech.

## What This Shows
- Live KPI cards: approval rate, delinquency, net loss, NPS
- Trend analysis: risk metrics and origination volume over 18 months
- Risk band segmentation: default rates across A/B/C/D bands
- Regional breakdown: loan volume vs default rate by city
- Interactive filters: date range and metric selector
- Raw data table with formatted financial figures

## Tools
Python · Streamlit · Plotly · pandas · Google Colab

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run dashboard.py
```
